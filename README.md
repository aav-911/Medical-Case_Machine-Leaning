# The Quality of Hospital Care | CSIS 344

## Inpatient Mortality Rate in Hospitals
Patients with different medical conditions still have risks of death while being 
in hospitals.
This can be caused by the quality of performance of the hospital.

## Data
This dataset shows how different hospitals perform better than others 
based on both mortality rate and quality throughout the years.

This analysis can be seen through the following categories
* County
* Hospital
* Cases of each condition
* Deaths of patients with each condition
* Type of medical condition

Link: [Data.gov](https://catalog.data.gov/dataset/california-hospital-inpatient-mortality-rates-and-quality-ratings-8e21f)

## Experimental Design
For X value, I will be using the following measures:
* Number of cases of each conditions
* Number of deaths of patients with each condition

For y value, I will use the following label:
* Type of medical condition
